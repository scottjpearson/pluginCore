Form

Used to assist with repopulating forms..

Dependencies:  None

Features:

  • repopulate_checkbox_radio
  • repopulate_text

repopulate_checkbox_radio( $name = '', $value='', $post = array(), $return = TRUE )

Takes the $name of a checkbox or radio button and the $value set for the input. Looks in the $post array to see if $post[$name] === $value if so it will mark the input as checked.

require_once '../Core/bootstrap.php';
$Core->Libraries(array( 'Form' ));

// Set the radio input patGender to checked if the value is 'Male'

<label> <input type="radio" name="patGender" value="Male" <?=Plugin\Form::repopulate_checkbox_radio('patGender','Male',$_REQUEST);?> /> Male </label>

repopulate_text( $name = '', $post = array(), $return = TRUE )

Takes the $name of a form field and an array of $post values, which should contain the $name as an array key. This function RETURNS the value of the array element found at key $name.

require_once '../Core/bootstrap.php';
$Core->Libraries(array( 'Form' ));

// Set the value of the patAge input to the value $_REQUEST['patAge']; if exists.

<input style="width: 100px;" type="text" class="form-control" name="patAge" placeholder="" value="<?Plugin\Form::repopulate_text('patAge',$_REQUEST);>" />
