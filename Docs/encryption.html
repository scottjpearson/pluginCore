Encryption Helper

Exposes functions to aid in encrypting/decrypting strings.

Functions:

  • encrypt
  • decrypt

encrypt

Used to encrypt a given string.

require_once '../Core/bootstrap.php';

$Core->Helpers(array('Encryption'));

$str = 'Text To Be Encrypted';

// Encrypt string
$encryptStr = encrypt( $str );

decrypt

Used to decrypt a given string.

require_once '../Core/bootstrap.php';

$Core->Helpers(array('Encryption'));

// Decrypt string
$str = decrypt( $encryptStr );
