MetadataCollection Class

A class for accessing the metadata for a group of metadata objects.

Usage

MetadataCollection is an extension of the ArrayObject and can be interacted with in a similar way, but it should only be used to hold Metadata objects. $metadataCollection = new MetadataCollection();
$metadataCollection[] = $metadata;
